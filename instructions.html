Instructions to Play Remember the Pattern

  • Press any key on the keyboard to start the game.
  • Game has four colored buttons. They are Green, Red, Yellow, Blue
  • Inital level of the game is one. A pattern is generated by the game after start of the game. Follow the pattern.
  • Example of the game pattern:
    Game: Red   User: Red
    Game: Blue   User: Red, Blue.
    Game: Yellow   User: Red, Blue,Yellow.
    so on....
